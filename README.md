# android-app

작은 규모의 식당을 위한 판매 및 재고, 상품 관리 POS 시스템 앱입니다.

## 목표

- 직원 4~5명이 사용하는 소형 식당 POS
- 스마트폰 기반으로 주문, 주방, 결제 기능 연동
- 한 대의 메인폰을 서버로 지정하여 내부 네트워크에서 동기화

## 주요 기능

1. 메뉴 주문
   - 고객 주문 접수
   - 주문 내역 공유 및 수정
2. 주방 연동
   - 주방 화면에 주문 전달
   - 조리 완료 상태 표시
3. 결제 관리
   - 현금/카드 결제 처리
   - 영수증 출력 또는 전송
4. 재고 및 상품 관리
   - 상품 등록/수정/삭제
   - 재고 수량 관리
   - 재고 부족 알림

## 시스템 구성

- 메인폰: 서버 역할
- 다른 직원용 폰: 클라이언트 역할
- 로컬 네트워크 또는 Wi-Fi 기반 동기화

## 향후 개발 계획

- Android 앱 기본 UI/UX 설계
- 네트워크 통신 및 데이터 동기화 구현
- 주문/주방/결제 흐름 개발
- 재고 관리 백엔드 로직 추가
